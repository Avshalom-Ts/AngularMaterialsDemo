Start whith this Basics Comands : ( From the CLI / CMD )

  1. 1. Install Angular Globaly npm install -g @angular/cli
  2. 2. Go to the relevant folder to creat your project ng new "Your project-name"
  3. 3. Type in the CLI / CMD (To view your project in the browser) ng serve
  4. 4. If you want to add Angular Materials ng add @angular/material
  5. 5. The preferd way to use angular material is by seperated model : ng g m materials
